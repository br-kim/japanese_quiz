# Readme

일본어 퀴즈 웹 애플리케이션

### 사용 기술

- Backend
    - Python
    - FastAPI
- FrontEnd
    - Javascript
    - Bootstrap
- DB
    - PostgreSQL
    - Redis
- CI/CD
    - GitHub Actions
    - AWS CodePipeline

### 주요 기능

- 로그인 / 회원가입
    - Google 소셜 로그인
- 랜덤 퀴즈
    - 랜덤하게 나오는 문자를 맞추는 퀴즈
- 테스트
    - 모든 문자가 한 번씩 나오는 테스트
    - 오답노트
        - 틀린 문자만 다시 테스트
- 자유게시판
    - 글 작성, 댓글 작성
- 채팅
    - 웹소켓을 사용한 채팅 기능
